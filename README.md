# Engine

An SDL3 engine template implementing GPU API

## Getting started

1. Create a new project from this template
2. Update git submodules by running following command from project root: `git submodule update --init --recursive`
3. Reload CMake project:
    - Generates the necessary build files
    - Copies assets from the `{project_root}/assets/output` to `{output_directory}/assets`
4. Build & Run project
